# Infrared Image Colorization and Enhancement for Improved Object Interpretation

## Overview
This project aims to improve the interpretation of infrared (IR) satellite imagery by enhancing image quality and generating realistic RGB representations from monochrome IR images.

The solution combines:
- Infrared Super-Resolution
- GAN-based IR → RGB Colorization
- Semantic Constraint Integration
- Improved Object Detection and Segmentation

## Problem Statement
Satellite infrared images suffer from:
- Low contrast
- Lack of semantic textures
- Difficult object interpretation during night-time and adverse weather.

Our solution addresses these challenges using Deep Learning and Remote Sensing techniques.

## Features
- High-resolution enhancement of IR images
- Realistic color prediction
- Semantic consistency preservation
- Artifact reduction
- Better downstream computer vision performance
